# manjaro-init

majaro-xfce init before install success

## init

```sh
$ git clone https://github.com/yiranzai/manjaro-init.git ~/.manjaro-init
$ cd ~/.manjaro-init
$ ./init.sh
```

or

```sh
curl -s  https://raw.githubusercontent.com/yiranzai/manjaro-init/master/init.sh | sh
```
